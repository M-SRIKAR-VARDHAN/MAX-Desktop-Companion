# Changelog

All notable changes to the MAX Desktop Companion project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - The Crash Landing

### Added
- **Core Physics Engine:** Allowed MAX to process desktop hitboxes, stand on taskbars, surf moving windows, ride cursors, and swing from window edges like pendulums.
- **Utility Modules:** One-click Screenshot, GIF Screen Recorder, Stopwatch, Timer, Color Picker, WiFi Password viewer, Local/Public IP fetcher, and Battery/Health monitors.
- **System Management Utilities:** Kill frozen processes, Uninstall apps from the registry, Windows App Arranger, Always-on-Top Pinner, Focus Mode, Temp File Cleaner, and Startup Manager.
- **Personality Core:** 13 dynamic facial expressions (happy, scared, dizzy, etc.), clipboard awareness (e.g., reacting to copied code), cursor dodge/pounce mechanics, and 3 AM philosophical wisdom quotes.
- **Secrets & Sandbox:** The Oracle terminal (accessed by typing `lore` or `wisdom` directly) and 7 cheat codes (`rave`, `giant`, `gravity0`, `clone`, `diwali`, `draw`, `magnet`).
- **The Lore Subsystem:** Full storyline integration of "The Null Void" and quantum tether mechanics embedded into core app interactions.

### Fixed
- SmartScreen flagging and Antivirus false positives documented in `README.md` for transparency.
- Fullscreen exclusive rendering logic: MAX steps aside automatically during borderless rendering and Netflix viewing to prevent overlap.
- Multi-monitor physics alignment adjustments (optimized for 100% Display Scaling).

### Security
- Created `SECURITY.md` for safe vulnerability reporting.
- Enforced zero background telemetry and 100% offline functionality.
