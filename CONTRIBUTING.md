# Contributing to MAX

Thanks for your interest in MAX!

## How to Help

### Report Bugs
1. Open a [Bug Report](https://github.com/M-SRIKAR-VARDHAN/MAX-Desktop-Companion/issues/new?template=bug_report.md)
2. Describe what happened and what you expected
3. **Attach your log file** from `%APPDATA%\ProjectMAX\max.log`

### Request Features
1. Open a [Feature Request](https://github.com/M-SRIKAR-VARDHAN/MAX-Desktop-Companion/issues/new?template=feature_request.md)
2. Describe the feature and why it would be useful

### Spread the Word
- Star this repo
- Visit and share our [Website](https://max-desktop-companion.vercel.app/)
- Share on social media
- Tell your friends

## Code Contributions
MAX is currently maintained by a small team. Source code is in a private repository.
If you're interested in contributing code, reach out to us via [Issues](https://github.com/M-SRIKAR-VARDHAN/MAX-Desktop-Companion/issues).

## Code of Conduct
Be kind. Be respectful. We're all here because we love desktop pets.
